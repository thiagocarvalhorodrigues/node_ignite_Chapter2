# node_ignite_Chapter2
Basic creation of a TypeScript application.
```
Using Express and nodemon
```

### dependencies 
```
node 14+
npm 7+
```
### How install
```
npm install
```
### How config 
```
|      name      | default |  type  | required |                            description                            |
| :------------: | :-----: | :----: | :------: | :---------------------------------------------------------------: |
|      PORT      |  9292   | number |   yes    |                     the port of endpoint API                      |


```
### How run
```
npm run start
```
### Port enable
```
define PORT in .env
```
### Endpoints
**Url** : `http://localhost:9292/`
